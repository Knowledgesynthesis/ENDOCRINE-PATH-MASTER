# **7. ENDOCRINE PATH REASONING FRAMEWORK**

1. Identify gland/site  
2. Evaluate architecture  
3. Assess cytology  
4. Look for invasion (capsular/vascular) where relevant  
5. Apply IHC conceptually  
6. Apply molecular context (high-level only)  
7. Integrate into broad diagnostic families  
8. Identify pitfalls and avoid overcalls  

Common pitfalls:  
- Misreading PTC-like nuclear features from artifacts  
- Calling hyperplasia adenoma in parathyroid  
- Confusing adrenal cortical carcinoma with oncocytic mimics  
- Overcalling pituitary hyperplasia as adenoma  
- NET grading confusion
